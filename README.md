# 🐍 贪吃蛇 H5 小游戏

一个简单的贪吃蛇网页游戏，可以直接部署到 AtomGit Pages。

## 游戏特性

- 🎮 经典贪吃蛇玩法
- 📱 响应式设计，支持移动端
- 💾 本地存储最高分
- ⚡ 随分数增加难度提升
- 🎨 渐变色精美界面

## 快速开始

1. 用浏览器打开 `index.html` 即可开始游戏

## 部署到 AtomGit Pages

### 前置要求
- 已注册 AtomGit 账号
- 已开通 AtomGit Pages 服务（需要申请审核）

### 部署步骤

1. **申请 AtomGit Pages**
   - 进入 AtomGit 开放市场
   - 找到 Atomgit Pages 应用并点击"免费使用"
   - 选择命名空间并填写申请信息
   - 等待运营审核通过

2. **创建公开仓库**
   - 在已安装 Pages 的命名空间下创建仓库
   - **注意：仓库必须设置为公开（公开仓库）**

3. **上传代码**
   - 将 `index.html` 上传到仓库根目录
   - 提交代码

4. **配置 Pages**
   - 进入仓库设置 → Pages
   - 选择要部署的分支（如 main/master）
   - 保存配置

5. **访问游戏**
   - 部署完成后会获得访问地址，格式如：
     `https://yourname.atomgit.net/your-repo/`

### 重要提示

- ❌ 私有仓库不支持 Pages 部署
- ❌ 仓库从公开转为私有会删除已部署的 Pages
- ✅ 根目录必须有 `index.html`
- ✅ 目前不支持不带二级目录的 Pages（即不支持 `xxx.atomgit.net`，必须是 `xxx.atomgit.net/repo`）

## 操作说明

### PC 端
- 方向键 ↑↓←→ 控制方向
- WASD 键也可以控制
- 空格键暂停/继续

### 移动端
- 使用屏幕上的方向按钮控制

## 游戏规则

1. 吃掉红色食物得分（每颗 +10 分）
2. 避免撞墙
3. 避免撞到自己的身体
4. 每得 50 分速度会加快

## 开发

纯 HTML/CSS/JavaScript 实现，无需构建工具，修改 `index.html` 即可自定义游戏。
