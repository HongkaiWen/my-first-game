# 🐍 贪吃蛇 H5 小游戏 v2.0

一个功能丰富的贪吃蛇网页游戏，支持多种趣味玩法和移动端优化。

## 游戏特性

### 🎮 核心玩法
- 经典贪吃蛇玩法
- 随分数增加难度提升
- 本地存储最高分

### ✨ 趣味功能
- 🌟 **特殊食物系统**
  - 红色普通食物：+10 分
  - 金色食物（10%）：+50 分
  - 蓝色减速食物（10%）：降低游戏速度
  - 紫色无敌食物（5%）：5秒无敌，可穿墙穿自己
- 🔥 **连击系统**：快速连续吃食物获得最高5倍分数倍率
- 🎆 **粒子特效**：吃食物时的爆炸效果
- 🔊 **音效系统**：使用 Web Audio API 生成游戏音效
- 🏆 **成就系统**：8个成就等待解锁

### 📱 移动端优化
- 大尺寸方向按钮（70x70px）
- 滑动手势控制（在游戏区域滑动）
- 触控振动反馈
- 响应式布局
- 防止双击缩放

### 🎨 视觉效果
- 渐变色精美界面
- 无敌状态蛇身发光闪烁
- 成就解锁动画通知
- 连击倍率显示

## 快速开始

1. 用浏览器打开 `index.html` 即可开始游戏

## 部署到 AtomGit Pages

### 前置要求
- 已注册 AtomGit 账号
- 已开通 AtomGit Pages 服务（需要申请审核）

### 部署步骤

1. **申请 AtomGit Pages**
   - 进入 AtomGit 开放市场
   - 找到 Atomgit Pages 应用并点击"免费使用"
   - 选择命名空间并填写申请信息
   - 等待运营审核通过

2. **创建公开仓库**
   - 在已安装 Pages 的命名空间下创建仓库
   - **注意：仓库必须设置为公开（公开仓库）**

3. **上传代码**
   - 将 `index.html` 上传到仓库根目录
   - 提交代码

4. **配置 Pages**
   - 进入仓库设置 → Pages
   - 选择要部署的分支（如 main/master）
   - 保存配置

5. **访问游戏**
   - 部署完成后会获得访问地址，格式如：
     `https://yourname.atomgit.net/your-repo/`

### 重要提示

- ❌ 私有仓库不支持 Pages 部署
- ❌ 仓库从公开转为私有会删除已部署的 Pages
- ✅ 根目录必须有 `index.html`
- ✅ 目前不支持不带二级目录的 Pages（即不支持 `xxx.atomgit.net`，必须是 `xxx.atomgit.net/repo`）

## 操作说明

### PC 端
- **方向键** ↑↓←→ 控制方向
- **WASD 键** 也可以控制
- **空格键** 暂停/继续

### 移动端
- **方向按钮**：使用屏幕上的大按钮控制
- **滑动手势**：在游戏区域滑动手指控制方向
- **触控反馈**：操作时有振动反馈

## 游戏规则

1. 吃掉食物得分（普通+10，金色+50）
2. 避免撞墙和撞到自己（无敌状态除外）
3. 每得 50 分速度会加快
4. 1.5秒内连续吃食物触发连击，分数倍率最高5倍
5. 特殊食物有不同的效果：
   - **减速食物**：暂时降低游戏速度
   - **无敌食物**：5秒内可以穿墙和穿过自己
